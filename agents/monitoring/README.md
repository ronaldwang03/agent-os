# Monitoring & Maintenance Agents

This directory contains documentation for agents that monitor systems and assist with maintenance tasks.

## Purpose

Monitoring & Maintenance agents help teams:
- Monitor system health and performance
- Detect and alert on issues
- Perform routine maintenance
- Analyze logs and metrics
- Provide insights and recommendations

## Agents in this Category

Currently, no agents have been documented in this category. To add an agent:

1. Use the [agent template](../templates/AGENT_TEMPLATE.md)
2. Create a new file in this directory
3. Update the [AGENTS.md](../../AGENTS.md) file
