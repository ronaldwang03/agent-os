# Planning & Requirements Agents

This directory contains documentation for agents that assist with project planning and requirement gathering phases of the SDLC.

## Purpose

Planning & Requirements agents help teams:
- Gather and analyze requirements
- Create project plans and roadmaps
- Estimate effort and resources
- Define project scope and objectives
- Facilitate stakeholder communication

## Agents in this Category

Currently, no agents have been documented in this category. To add an agent:

1. Use the [agent template](../templates/AGENT_TEMPLATE.md)
2. Create a new file in this directory
3. Update the [AGENTS.md](../../AGENTS.md) file
