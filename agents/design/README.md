# Design & Architecture Agents

This directory contains documentation for agents that help with system design and architectural decisions.

## Purpose

Design & Architecture agents help teams:
- Design system architecture
- Make technology stack decisions
- Create design patterns and best practices
- Document architectural decisions
- Review and validate designs

## Agents in this Category

Currently, no agents have been documented in this category. To add an agent:

1. Use the [agent template](../templates/AGENT_TEMPLATE.md)
2. Create a new file in this directory
3. Update the [AGENTS.md](../../AGENTS.md) file
