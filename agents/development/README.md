# Development & Coding Agents

This directory contains documentation for agents that support code generation, refactoring, and development tasks.

## Purpose

Development & Coding agents help teams:
- Generate boilerplate code
- Refactor existing code
- Implement features
- Follow coding standards
- Automate repetitive coding tasks

## Agents in this Category

Currently, no agents have been documented in this category. To add an agent:

1. Use the [agent template](../templates/AGENT_TEMPLATE.md)
2. Create a new file in this directory
3. Update the [AGENTS.md](../../AGENTS.md) file
