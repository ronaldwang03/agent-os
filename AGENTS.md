## How to Update This Document

When adding a new agent:

1. Create agent documentation in the appropriate `agents/` subdirectory
2. Add an entry in the relevant category section above
3. Update the summary count table
4. Follow this format for agent entries:

```markdown
### [Agent Name](link-to-agent-doc)

**Status:** [Status]  
**Owner:** [Team/Person]  
**Description:** Brief description of what the agent does  
**Last Updated:** YYYY-MM-DD
```
